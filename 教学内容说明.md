# 教学内容管理功能说明

## 📝 重要提示

### 示例内容不再写在主代码中！

从 v2.0 开始，示例教学内容**不再写在 main.py 中**，而是通过以下两种方式添加：

1. **运行初始化脚本**（推荐首次使用）
2. **通过 Web 界面手动添加**（推荐日常使用）

这样做的好处：
- ✅ 主代码更简洁，易于维护
- ✅ 不会在每次启动时重复添加数据
- ✅ 可以灵活控制何时添加示例内容
- ✅ 方便修改和更新内容

## 🚀 使用指南

### 方法 1：运行初始化脚本（快速添加示例内容）

```bash
# 1. 清空现有数据（可选，如果需要重新开始）
.venv\Scripts\python.exe clear_contents.py
# 输入 yes 确认

# 2. 初始化示例内容
.venv\Scripts\python.exe init_sample_contents.py
```

### 方法 2：通过 Web 界面添加（推荐）

1. **登录系统**
   - 访问：http://localhost:5173
   - 账号：`admin` / `DEFAULT_ADMIN_PASSWORD`（或 `init_db.py` 输出的随机密码）
   - 如需测试统一口令：运行 `scripts/set_test_passwords.py`

2. **进入教学资源页面**
   - 点击"📚 教学资源"按钮

3. **添加内容**（教师/管理员）
   - 点击"+ 新建内容"
   - 填写标题、选择分类
   - 选择类型（文章/视频/图片/PDF）
   - 输入内容（支持 Markdown 格式）
   - 勾选"发布"
   - 点击"保存"

4. **管理分类**（教师/管理员）
   - 点击"+ 新建分类"
   - 输入分类名称和描述
   - 可选择父分类（创建子分类）

## 🔧 常见问题

### Q: 默认账号密码不是 admin/admin，如何登录？
A: 当前初始化策略为“环境变量优先 + 随机强口令兜底”。请按以下顺序处理：
   1. 查看首次执行 `init_db.py` 时控制台输出的默认密码
   2. 或设置 `DEFAULT_ADMIN_PASSWORD` 后重新初始化
   3. 或执行 `scripts/set_test_passwords.py` 统一测试口令

### Q: 为什么教师看不到某些未发布内容？
A: 教师只能管理和发布自己创建的内容；管理员可管理全部内容。

### Q: 统计卡片里的数字怎么理解？
A: 页面已区分“人数”和“比例”。
   - 人数：实际学习人数/完成人数
   - 比例：完成率（%）

### Q: 如何修改已添加的示例内容？
A: 在教学资源页面，找到对应内容，点击"编辑"按钮即可修改。

### Q: 如何删除示例内容？
A: 点击内容卡片上的"删除"按钮。或者运行 `clear_contents.py` 清空所有数据。

### Q: 学生可以看到所有内容吗？
A: 学生只能看到"已发布"状态的内容。教师和管理员可以看到所有状态的内容。

### Q: 如何批量导入内容？
A: 目前支持逐个添加。如需批量导入，可以：
   1. 修改 `init_sample_contents.py` 脚本
   2. 添加更多内容数据
   3. 运行脚本导入
